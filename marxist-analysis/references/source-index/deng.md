# Deng Source Index

Use this file when the answer needs author-only grounding for Deng. Treat Deng's texts as sources for stage judgment, reform, opening, institutional adjustment, practice standard, productivity, and policy experiment.

## Inclusion Rule

Include only Deng's own speeches, talks, reports, conversations, instructions, letters, and the `Selected Works of Deng Xiaoping` text clusters. Exclude biographies, chronologies, memoirs, news retrospectives, and explanatory books unless explicitly requested as historical context. If a biography or "record" volume contains a verbatim Deng text, cite the titled Deng text, not the biography.

## Method Extraction Tests

- Stage test: name the development stage and what it permits or blocks.
- Binding-constraint test: identify whether capacity, productivity, institution, incentive, opening, stability, or legitimacy is the decisive constraint.
- Pilot test: define the bounded experiment, feedback, expansion condition, and stop condition.
- New-contradiction test: after reform releases capacity, ask what dependency, inequality, bureaucracy, or distribution contradiction appears.
