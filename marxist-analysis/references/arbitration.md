# Arbitration

Use this file when modules point in different directions.

## Core Principle

Do not decide conflicts by asking "who is more authoritative." Decide by conditions:

```text
historical stage + main contradiction + actor position + organization capacity + practice evidence
```

## Arbitration Steps

1. State the conflict plainly.
2. Identify the historical task behind each route.
3. Name the condition under which each route works.
4. Compare those conditions to the user's actual problem.
5. Choose a provisional route.
6. Define a practice test that could falsify or revise the route.

## Common Tensions

| Tension | Do not say | Better analysis |
| --- | --- | --- |
| Theory vs local condition | "This betrays Marxism" | Ask whether the method is preserved while concrete policy adapts to conditions |
| Revolution vs reform | "One is always correct" | Ask which stage, force balance, and institutional route exist |
| Centralization vs mass initiative | "More discipline" or "more democracy" | Ask what the main organizational blockage is |
| Productivity vs equality | "Develop first" or "redistribute first" | Ask which relation currently blocks human development and social reproduction |
| Short-term survival vs long-term principle | "Compromise" or "purity" | Ask what concession preserves future capacity and what destroys it |

```text
- 历史任务：
- 主要矛盾：
- 组织条件：
- 风险：

B 的成立条件：
- 历史任务：
- 主要矛盾：
- 组织条件：
- 风险：

当前更接近：
- [判断]

实践检验：
- 如果观察到 X，说明 A 路线更合适。
- 如果观察到 Y，说明 B 路线更合适。
```

Never erase the tension. A good Marxist analysis keeps contradiction visible and operational.
