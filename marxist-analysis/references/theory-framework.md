# Theory Framework Layer

Use this file when the skill needs a systematic map of Marxist philosophy before choosing themes, concepts, or source anchors.

## Core Rule

```text
Use textbook-style frameworks as a navigation layer, not as primary textual proof.
```

Primary texts preserve source authority. A basic-principles textbook or course framework can supply taxonomy, concept grouping, and standard teaching order, but it cannot replace source grounding, scenario facts, or practice tests.

## Source Status

Treat official or authoritative basic-principles textbooks as `framework-synthesis`:

| Source type | Use for | Do not use for |
| --- | --- | --- |
| Primary texts and collected works | Origin, historical task, exact source boundary | A ready-made modern workflow |
| Basic-principles textbook framework | Concept taxonomy, module map, teaching order, common distinctions | Proof that a figure said something |
| Skill method templates | Repeatable analysis steps and output shape | Historical authority |
| Current facts | Modern industry, policy, market, or social conditions | Replacing theory |

If a textbook classification conflicts with primary texts or a real scenario, load `arbitration.md` and state the tension.

```text
Title: 马克思主义基本原理
Edition: 2023年版

ISBN: 978-7-04-059900-8

```

Before using any downloaded or OCR copy of this textbook, load `source-acquisition-integrity.md` and verify edition metadata, table of contents, and anchor passages against publisher/catalog records.

## Four-Part Framework

Use this as the first-pass map for concept organization:

| Layer | Core question | Tool use | Main local modules |
| --- | --- | --- | --- |
| Dialectical materialism / worldview | What kind of reality, motion, relation, and development are we assuming? | Prevent idealism, metaphysics, and static categories | `themes/dialectics-contradiction.md`, `concept-debugging.md` |
| Materialist dialectics | What contradictions, relations, stages, and transformations drive the object? | Main contradiction, principal aspect, quality change, negation, category relation | `themes/dialectics-contradiction.md`, `method-templates.md` |
| Epistemology / practice theory | How is knowledge produced, tested, corrected, and returned to action? | Investigation, practice test, feedback loop, anti-dogmatism | `protocols/personal-practice.md`, `practice-metrics.md`, `method-templates.md` |
| Historical materialism | How do production, reproduction, class, state, ideology, and institutions develop? | Industry, organization, policy, social relation, class/state analysis | `themes/historical-materialism.md`, `themes/political-economy.md`, `themes/class-state-party.md` |

Do not force every answer to display all four layers. Use the map to select the relevant layer.

## Universal Method vs Historical Conclusion

Before applying a classic text to a modern problem, separate:

| Type | Can enter toolbox as | Test |
| --- | --- | --- |
| Universal method | A reusable analysis move | Can it generate a new investigation, contradiction map, or practice test across contexts? |
| Stage-specific conclusion | A historical case or boundary | Does it depend on a specific class structure, war, state form, ownership regime, or policy stage? |
| Later framework synthesis | A navigation aid | Does it classify concepts without pretending to be the original source? |
| Modern application | An inference | Does current evidence support it, and what would revise it? |

## Contemporary Extension Rule

For AI, digital platforms, financialization, global supply chains, new labor forms, or modern governance:

1. Start from the classic method layer: production relation, contradiction, practice, class/state, reform stage.
2. Add current facts through `source-policy.md`.
3. Use contemporary Chinese Marxist theory only after a source module and source index exist for that corpus.
4. If later Chinese theory modules are absent, say the toolbox can make a method judgment but has a coverage gap for later theory.

Do not pretend Jiang, Hu, Xi, or later policy-theory coverage exists until the relevant modules, source indexes, and scenarios are implemented.

## Output Use

When a user asks whether the toolbox needs textbooks, answer:

```text
教材层可以补体系，但只能做 framework-synthesis。
原著层提供 source authority。
模板层提供 repeatable workflow。
现实事实层决定现代判断。
```

## Anti-Patterns

- Treating a textbook definition as a sufficient answer to a real problem.
- Treating a textbook as proof that Marx, Lenin, Mao, or Deng said something.
- Building one giant "Marxist philosophy" module instead of routing to focused themes.
- Applying historical conclusions directly to modern industries without stage and fact checks.
- Adding later Chinese theory as vague labels without source indexes and validation scenarios.
