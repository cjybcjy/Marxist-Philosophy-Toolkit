# Mass Line

## Use For

Use this theme for leadership, user/community feedback, organizing, coalition building, research, and converting scattered experience into a line of action.

## Core Lens

The mass line is not "ask people what they want and obey." It is:

```text
from the masses -> synthesize -> return to the masses -> test in practice
```

## Source Anchors

- Mao's leadership-method writings for "from the masses, to the masses."
- Mao's investigation writings for direct experience and concrete conditions.

Use these anchors to convert lived experience into a line of action. Do not treat mass line as passive opinion polling.

## Analysis Moves

1. Gather direct experience from affected people.
2. Separate lived facts from noise, fear, fashion, and hearsay.
3. Synthesize a line that expresses real interests and feasible direction.
4. Return it as a practical action, not as propaganda.
5. Observe whether people can use it, improve it, or reject it.

## Common Misuse

- Mistaking popularity for correctness.
- Mistaking elite theory for reality.
- Turning "mass line" into survey collection only.
- Returning slogans instead of usable action.

## Boundary Test

If the answer never returns a usable action to the people who supplied the experience, it is not mass line.

## Practical Prompt

```text
谁有直接经验？
他们反复遇到什么阻力？
哪些说法只是情绪，哪些指向结构？
我能返回给他们的最小可用方案是什么？
```
