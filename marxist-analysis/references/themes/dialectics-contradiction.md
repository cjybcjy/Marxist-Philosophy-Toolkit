# Dialectics And Contradiction

## Use For

Use this theme when the user asks about main contradiction, strategic priority, internal tension, route choice, or why a situation keeps changing.

## Core Lens

Complex situations contain multiple contradictions. One contradiction usually has decisive weight in a given stage. Inside it, one aspect may dominate.

## Source Anchors

- Mao's contradiction writings for main contradiction, principal aspect, and transformation.
- Marxist dialectical tradition for contradiction as movement, not static opposition.

Use these to identify the decisive relation in a concrete stage. Do not use them to force every problem into binary opposition.

## Analysis Moves

1. List the visible tensions.
2. Separate surface symptoms from structural contradictions.
3. Name the main contradiction for the current stage.
4. Name the main aspect of that contradiction.
5. Identify conditions under which the contradiction can transform.
6. Translate the analysis into a focused action.

## Main Contradiction Test

A contradiction is likely "main" when:

- solving it changes several other problems,
- avoiding it keeps producing repeated symptoms,
- resources, incentives, and conflict cluster around it,
- a small test can reveal whether it is decisive.

## Common Misuse

- Calling every conflict the main contradiction.
- Treating contradiction analysis as a slogan.
- Ignoring time: a main contradiction can change by stage.
- Jumping to struggle language before investigating conditions.

## Boundary Test

If solving the named contradiction would not change the user's other major problems, it is probably not the main contradiction.
