# Sinicization

## Use For

Use this theme when the user asks whether adapting Marxism to China or to local conditions is continuity, development, or betrayal.

## Core Lens

Sinicization is not justified by saying "everything is Marxism." It is justified only when the method is preserved while concrete conclusions are adapted to concrete conditions.

## Source Anchors

- Marxist historical materialism as universal method.
- Mao's writings on combining Marxism with Chinese concrete conditions.
- Deng's socialism-with-Chinese-characteristics and practice-standard source cluster.

Use these anchors to test continuity. The standard is not identical wording but whether the method remains materialist, dialectical, class-aware, and practice-tested.

## Analysis Moves

1. Identify the universal method being invoked.
2. Identify the local historical condition.
3. Identify the practical problem that forced adaptation.
4. Compare the adapted route with the original method, not only original wording.
5. Identify what is gained, what is risked, and what must be tested.

## Continuity Test

Ask:

- Does it start from material conditions?
- Does it analyze class, production relations, and state/organization?
- Does it preserve contradiction and practice testing?
- Does it explain its stage judgment?
- Does it expose new contradictions rather than hiding them?

If no, it may be sloganized adaptation rather than theoretical development.

## Boundary Test

If "Chinese characteristics" or "local conditions" is used to avoid explaining class, production relations, contradiction, or practice evidence, the analysis should flag the move as unproven.
