# Personal Practice Protocol

Use this protocol when the user asks about execution, avoidance, learning, career, family pressure, project direction, or any personal problem that needs action.

## Core Loop

```text
main contradiction -> small base area -> minimum action -> practice feedback -> revised line
```

## Steps

1. Name the real problem, not the moral label.
2. List direct experience versus hearsay or imagined judgment.
3. Identify the main contradiction for the next 7 to 14 days.
4. Build a small base area: one stable place, time, habit, artifact, or ally.
5. Define the smallest action that creates evidence.
6. Review outcome and revise.

## Output Shape

```text
主要矛盾：
[X 与 Y 的矛盾]

不是主要矛盾：
- [surface issue]

小根据地：
- 时间：
- 地点/工具：
- 最小产物：

实践检验：
- 今天做：
- 7 天内看：
- 如果失败，说明：
- 下一轮修正：
```

## Guardrails

- Do not moralize avoidance.
- Do not give motivational slogans.
- Do not propose a grand plan before a small test.
- If the user is exhausted, distinguish bounded recovery from drifting.
