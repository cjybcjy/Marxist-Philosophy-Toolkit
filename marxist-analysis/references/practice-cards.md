# Practice Cards

Use these cards when the user wants action, diagnosis, training, or a decision. Fill one card rather than producing a broad essay.

## Card Selection

| User Need | Use Card |
| --- | --- |
| "What is the main contradiction?" | Main Contradiction Card |
| "What should I do today?" | Small Practice Card |
| "Who benefits or blocks?" | Interest Structure Card |
| "How do I investigate this?" | Mass Investigation Card |
| "Which route should we choose?" | Route Experiment Card |
| "How do we review results?" | Practice Review Card |

## Main Contradiction Card

```text
现象：
- [visible symptoms]

候选矛盾：
1. [A vs B]
2. [C vs D]

主要矛盾判断：
- 当前阶段最牵动全局的是：
- 主要方面是：

为什么不是其他矛盾：
- [surface issue] 只是表现，因为：

转化条件：
- 如果观察到 X，改判为：
```

## Output Rule

When using a card, keep the final answer compact. A good practice answer should make the next action easier to start, not make the theory feel larger.
